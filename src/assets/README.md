# Marenyo FC Assets Folder ⚽🍊

This directory contains the official images, club crests, and media assets for Marenyo FC.

## Uploading Your Own Images

You can place your real images here or inside the `public/assets/` directory:

1. **Official Club Logo**:
   - Replace `public/assets/club-logo.svg` or add `public/assets/club-logo.png`
   - Recommended size: 500x500 px (transparent PNG or SVG)

2. **Squad & Player Photos**:
   - Drop player photos into `public/assets/players/` or paste any direct URL / local path into the **Admin Portal > Player Roster** tab.

3. **Match Kit Images**:
   - Drop kit images into `public/assets/kits/` or enter custom image URLs in the **Admin Portal > Kits** tab.

4. **News & Announcement Banner Images**:
   - Drop news graphics into `public/assets/news/` or enter custom image URLs in the **Admin Portal > News** tab.
